Python Equivalent

import requests
import re

def main():
    url = "https://example.com"
    print(f"Fetching {url}")
    
    try:
        r = requests.get(url)
        
        if r.status_code != 200:
            print(f"Error: {r.status_code}")
            return False
        
        c = r.text
        p = []
        
        m = re.findall(r"
(.*?)
", c, re.DOTALL)
        
        for match in m:
            p.append(match)
        
        print(f"Found {len(p)} paragraphs")
        return True
    
    except Exception as e:
        print(f"Error occurred: {e}")
        return False

if __name__ == "__main__":
    main()

Getting Started

1. Installation

git clone https://github.com/APiTJLillo/Anarchy-Inference.git
cd Anarchy-Inference
pip install -r requirements.txt

Clone the repository and install dependencies to get started with Anarchy Inference.

2. Create Your First Program

// hello.ai
m{
  main(){
    print("Hello, Anarchy Inference!");
    return 0;
  }
}

Create a simple hello world program to test your installation.

3. Run Your Program

python anarchy.py hello.ai

Run your program using the Anarchy Inference interpreter.
